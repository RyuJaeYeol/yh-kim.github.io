> 인텔리제이에서 개발할 때 자주 사용하는 단축키를 정리합니다

## Code

* Delete unused code.
> Ctrl Alt O

* Show name,type of parameter
> Ctrl P

* Move to up/down by class, attribute
> Alt up/down

* Move to left/right by unit
> Ctrl left/right



## Search

* Search Everywhere
> Shift Shift

* Search action, option
> Ctrl Shift A

* Search file name
> Ctrl Shift N

* Search class name
> Ctrl N




## Window

* Hide/Restore All windows
> Ctrl Shift F12

* Recent File
> Ctrl E

* Switcher
> Ctrl Tab

* Show Windows
> Alt Alt

* tab
> Project - Alt 1<br>
> Structure - Alt 7
